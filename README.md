# Netflix Data Analysis

## Project Overview
This project analyzes Netflix movies and TV shows using Python for data cleaning
and exploratory data analysis, and Power BI for building an interactive dashboard.

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Google Colab
- Power BI

## Dataset
- Netflix Movies and TV Shows dataset
- Cleaned and processed using Python

## Power BI Dashboard
The dashboard shows:
- Total titles and countries
- Movies vs TV Shows distribution
- Top countries by content
- Release year trends
- Interactive slicers for country and title

## Key Insights
- Netflix content increased significantly after 2015
- Movies dominate Netflix content
- The United States is the top content-producing country
